1.What is the difference between block tags and inline tags.

= Block tags like H1, p, h2, always starts a new line and takes up the full width. While Inline tags like span or a only take much space as needed and does not start a new line

2.What is the margin, border, and padding?

= Margin is the space outside the element's border. Border is the line surrounding the element and padding is the space between the content and it's border.

3. What are units of measurement you can use in CSS?

= units of measurement we have used so far are, px, in, cm, and %

4. When would you use a figure tag?

= It is used to represent self-contained content like images.
